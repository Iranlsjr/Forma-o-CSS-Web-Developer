Pseudo-Elementos

Usando para adicionar elementos extras na página, sem a necesssidade de declará-lo no HTML, ou para selecionar um parte de algum elemento.

Identificando por ::Palavra-chave

Pseudo-Elementos - ::first-letter

Seleciona a primeira letra da primeira linha

Propriedades disponíveis:

  • Color
  • background
  • margin
  • padding
  • border
  • font
  • Propriedades de texto
