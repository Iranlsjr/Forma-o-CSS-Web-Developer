Seletores CSS

Bem-Vindo!

No CSS nós possuímos os seletores por ID

  • Seletor por tipo: Busca elementos por uma tag HTML especifica
  • Seletor por ID: Busca elementos através de atributo "id"
  • Seletor por Classe(.): Busca elementos através do atributo "class"
  • Seletor Universal(*): Seleciona todos os elementos HTML
  • Seletor de Atributo ([atrib]): Seleciona elementos que possuem um atributo especifico em nosso documento HTML e conseguimos também buscar atributos com um valor específico
