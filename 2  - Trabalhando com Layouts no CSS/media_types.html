Type @media All: Corresponde a todoas os dispositivos

Type @media print: Corresponde a documentos que são uma pré-visualização de impresão, ou em qualquer mídia que será voltada para imprimir

Typee @media screen: Corresponde a dispositivos com telas integradas

Type @media Speech: Corresponde a dispositivos que leem o conteúdo de forma audível, como um leitor de tela por exemplo
